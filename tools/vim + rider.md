# jet rider 🏇🏻

`Using AutoHotKey to substitute alt comma and dot for left and right 🏹s.`

### Current AutoHotKey movement

- `alt + j/k` movement down/up
- `alt + ctrl + j/k` move line down/up
- `alt + shift + j` copy line down
- `ctrl + shift + j/k` move selection up/down (ctrl + shift fakes alt)
- `ctrl + shift + alt + j/k` add cursor up/down
- `alt + ,/.` movement left/right

### Code

- 🔮 `ctrl + shift + d` -> *Add selection for next occurence.*
- 🦜 `alt + shift + right/left 🏹`-> Extend/Shrink Selection
- `ctrl + .` -> Show Context Actions (open error fixing menu, refactoring menu)
- `ctrl + shift + [ or ]` -> Fold/Unfold given function.

### Locations, files, actions

- Basic Search
	- `ctrl + p` -> Go to file...
	- `ctrl + shift + p` -> Search for Action...
	- 🔎 `ctrl + shift + f` -> Find in files
- Recents
	- 🦜 `ctrl + e` -> Recent files
	- 🏹 `ctrl + e + shift` -> Recent locations
- Editor Windows
	- `ctrl + b` -> Toggle file explorer tab
### Tabs

- Open/Close
	- `ctrl + w` Close Tab
	- 🪶 `ctrl + shift + t` Reopen Closed Tab
- Move around
	- 🪶 `ctrl + shift + m` Next tab splitter.
	- 🪶 `ctrl + shift + ,/.` Next/Previous Tab
	- 🦜 `ctrl + k, ctrl + q` -> Go to last edited location
