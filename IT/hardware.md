# Grundwissen

## Computerteile

***Was ist ein Motherboard?***
- **Hauptplatine eines Computers**
- Verbindet **CPU, RAM, Speicher, Grafikkarte, Peripherie** miteinander
- Enthält **Chipsatz, Anschlüsse und Steckplätze**

***Was ist RAM?***
- *Random Access Memory* 
- *Temporärer **Arbeitsspeicher***
	- In dem Daten und Programme während der Nutzung liegen.
- ***Flüchtig***
	- Inhalt geht *beim Ausschalten* verloren.

***Was ist eine Festplatte?***
- **Festplatte (HDD – Hard Disk Drive)** ist ein **dauerhafter Speicher** für Daten und Programme.
	- Speichert Informationen **magnetisch auf rotierenden Scheiben**.
- **Nichtflüchtig:** Daten bleiben auch nach Ausschalten erhalten.
- Alternative: **SSD (Solid State Drive)** – speichert Daten **elektronisch**, schneller als HDD, keine beweglichen Teile.

***Storage vs Memory***

|Begriff|Erklärung|
|---|---|
|Storage (Speicher)|Festplatten, SSDs – **dauerhafte Speicherung** von Daten|
|Memory (RAM)|Arbeitsspeicher – **temporäre Speicherung** für laufende Programme|

***Was ist die CPU?***
- **Central Processing Unit (Prozessor)** = „Gehirn des Computers“
- Führt **Rechenoperationen und Befehle** aus
- Steuert **Programme, Threads und Datenfluss**

***Was ist eine Grafikkarte?***
- **GPU (Graphics Processing Unit)** verarbeitet **Bilder, Videos, 3D-Grafik**
- Entlastet die CPU bei grafikintensiven Aufgaben
- Kann **integriert (onboard) oder dediziert** sein

***Was ist Virtualisierung?***
- **Erstellung virtueller Ressourcen** (Server, Speicher, Betriebssysteme) auf einer physischen Hardware
- **Mehrere virtuelle Umgebungen** können gleichzeitig laufen
## 🔹 Unterschied zwischen Prozess und Thread

- **Prozess:** eigenständiges Programm in Ausführung, hat **eigenen Speicherbereich**
    
- **Thread:** leichtgewichtiger Teil eines Prozesses, teilt **Speicher und Ressourcen** des Prozesses
    
- **Merksatz:** Threads laufen innerhalb von Prozessen und sind schneller zu erstellen als ganze Prozesse.
    

## 🔹 Was ist eine VM (Virtuelle Maschine)?

- **Virtuelles System**, das wie ein eigenständiger Computer läuft
    
- Läuft **innerhalb eines Hypervisors** auf physischer Hardware
    
- Kann **Betriebssysteme unabhängig vom Host** ausführen
